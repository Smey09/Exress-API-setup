## How to config it

     - Create :  .env.local
        * build/configs/.env.local

    - Create :  .env.development
        * src/configs/.env.development

## in side the file, you can put :

    - PORT=your_port
    - NODE_ENV=development
    - MONGODB_URL=mongodb+srv://<user_name>:<password>@bookscluster.xfo1s.mongodb.net/...
